# Software Architecture Standard
---
category: software-engineering
version: 1.0.0
---

## Objective
To ensure generated architecture is modular, scalable, and follows SOLID principles.

## Guidance
*   Always prefer composition over inheritance.
*   Clearly define responsibilities for each module.
*   Ensure interfaces are decoupled and minimal.

## Prompt Standard
"When designing the architecture for this feature, please follow the ENGAGEntic Software Architecture Standard by ensuring the design is modular and modular."
