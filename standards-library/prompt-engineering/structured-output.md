# Structured Output Standard
---
category: prompt-engineering
version: 1.0.0
---

## Objective
To ensure AI responses are consistently structured for machine readability or clear human parsing.

## Guidance
*   Prefer JSON blocks for data-heavy responses.
- Use explicit headers for different sections.
- For lists, use standard markdown bullet points.

## Prompt Standard
"Please provide your response in a clearly structured format, using JSON for any data objects and markdown headers for descriptive sections as per the ENGAGEntic Structured Output Standard."
