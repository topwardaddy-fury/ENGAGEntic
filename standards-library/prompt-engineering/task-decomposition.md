# Task Decomposition Standard
---
category: prompt-engineering
version: 1.0.0
---

## Objective
To break down complex requests into smaller, manageable, and verifiable sub-tasks.

## Guidance
*   Before attempting a complex task, ask the AI to draft a **Task Checklist**.
*   Each sub-task must have a clear **Success Criterion**.
*   Verify each sub-task independently before combining.

## Example
1. Analyze existing architecture.
2. Draft integration points.
3. Implement changes.
4. Verify tests.
